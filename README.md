# Sorting Visualizer
**Short description:** A simple interactive Sorting Algorithm Visualizer that demonstrates how different sorting algorithms work by animating array operations.
## Features
- Visualize common sorting algorithms (e.g., Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, Quick Sort).
- Adjustable array size and speed.
- Step-by-step animations to understand comparisons and swaps.
## Quick start
Open the project files or run the provided scripts. If unsure, check the repository root for `index.html` or `package.json`.

## Usage
- Choose an algorithm from the UI.
- Adjust array size or speed controls.
- Click **Start** to see the animation.
- Use **Shuffle/Reset** to test different arrays.
